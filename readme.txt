Universal Selector:
* {
  font-family: "Outfit", sans-serif;
  margin: 0;
  padding: 0;
}
Applies styles to all elements in the page.

Element Selectors:
body, h1, h2, p, img, a
Target individual HTML elements and apply styles

ID Selector(#profile-pic):
#profile-pic {
  display: flex;
  justify-content: center;
}
Targets a specific element with the ID profile-pic to center an image horizontally.

Class Selectors (.main-div, .sub-div, .contact, .list, .line)
.main-div, .sub-div, .contact, .list, .line
Applied styles to groups of elements for layout (.main-div, .sub-div), contact section (.contact), list style (.list), and  dividers (.line).

Attribute Selector (a[href]):
To target anchor tags having href attribute.

Descendant Selector (div p, div img, div h2):
Targets paragraph (p), image (img), and heading (h2) elements that are inside a div.

Group Selector(h1,h2):
Used to add styles to common elements, or classes.
